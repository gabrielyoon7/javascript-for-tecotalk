### 실행 방법

정상적인 실행을 위해서 vscode의 Code Runner 설치 후, 개별 파일을 run code 해주셔야 합니다.

```
설치가 되어있음에도 실행이 되지 않는 경우
환경설정에서 code-runner.run에 체크
```
